## TownySearch - Плагин для Paper, который добавляет к Towny удобную систему поиска городов для новичков на сервере.
Теперь игрок может всего одной командой отправить запрос на вступление в город всем мэрам! Также добавлена возможность вносить игроков в чёрный список, чтобы предотвратить спам.

### ✅ Поддержка цветовых тегов MiniMessage
### ⚙️ Полностью настраиваемый через конфигурацию
## Команды:
| Команда | Инфо |
| :--- | :--- |
| `/townsearch`<br>*(алиасы: `/tsearch`, `/townask`, `/task`)* | Отправка объявления о поиске города. **Обратите внимание:** эту команду могут использовать только игроки, не состоящие в городе! |
| `/townsearch ignore` | Открывает список игнорируемых вами игроков. |
| `/townsearch ignore <число>` | Открывает конкретную страницу игнор-листа. |
| `/townsearch ignore add/remove <игрок>` | Добавить или убрать игрока из игнор-листа. |
| `/townsearch reload` | Перезагрузить конфигурацию плагина. |

## Установка:
Скачайте Towny
Скачайте плагин.
Выключите сервер
Переместите файлы .jar в папку /plugins/ вашего сервера.
Запустите сервер.
Готово.
